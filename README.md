# SmartRubbishBin（智能倾倒垃圾桶）
 
  
  
### 仓库结构（暂定）

- `ros`文件夹存放与ROS相关的代码
- `CarModel`文件夹存放小车SolidWorks模型
- `Tutorial`中记录一些学习笔记
- `Makefile`配置自动构建或执行自动化任务
- `.gitignore`中填写不需要被git管理的文件/文件夹


### 开发环境配置

- ROS1-Noetic 完整版（带Gazebo等）
- SolidWorks 2021 + SW2URDF
- Python3
